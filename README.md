# AI 剪刀石頭布大賽 🖐️✌️✊

這是一個基於 MediaPipe Hands 實作的網頁視覺辨識遊戲。玩家可以使用手勢與電腦進行剪刀石頭布對決。

## 🚀 功能特點
- **視覺辨識**：即時手部骨架追蹤。
- **視窗化設計**：支援手機與電腦瀏覽器。
- **手勢控制**：
  - 🖐️ **布**：開始遊戲 / 繼續下一局 / 重新開始。
  - ✌️ **剪刀** / ✊ **石頭** / 🖐️ **布**：遊戲出拳。
  - 👍 **讚**：結算當前戰績。

## 📱 手機遊玩注意事項
1. 必須使用 **HTTPS** 環境開啟（GitHub Pages 預設已開啟）。
2. 進入網頁後，請務必點擊「允許」存取攝影機。
3. 手機建議橫屏或將手機固定，確保手掌能完整出現在視窗中。
